## Install on Termux

### Requirements

    libyaml-cpp >= 0.8
    boost >= 1.88.0

### Dependencies
Install the dependencies
```bash
pkg install boost libyaml-cpp
```

### Install
Download the nipovpn_VERSION_termux_aarch64 version from releases pages
```bash
pkg install ./nipovpn_VERSION_termux_aarch64.deb
```
